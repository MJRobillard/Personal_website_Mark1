Personal Projects

Spotify Car Radio Integration

My old car had a personal problem with Spotify but had no problem playing music from my iTunes. So using some libraries I found on GitHub, I wrote this to search for the songs in a playlist on Spotify in YouTube, download the audio and metadata, and put them into my iTunes.

Backend in Python for Chess

The code knows the options for how a piece can move, what a checkmate is, who's turn it is, offers options to the user, and pretty much everything except for a decent visual display.

Cal RSF Weight Room Optimization

I love when the weight room at the gym is nearly empty. The local gym posts a meter online showing how full it is, but they only offer a live update. So to find the best time to go to the gym, I built a web scraper using Selenium and Beautiful Soup to record live updates in a CSV every five minutes as a background process. Now I just have to let the program collect the data.

Cal Clubs Ranked

Those hiring and new students have little reference about how prestigious a tech club at UC Berkeley is. Sure they can look at the club's website, but it's hard to compare two different clubs. This project aims to solve that problem with a comprehensive website with ranking methodology and data of the clubs here at Cal.
